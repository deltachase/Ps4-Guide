    Wha' be homebrew?

    Homebrew means software that ain't made or authorized by th' dev (developer) o' th' console. This lets ye play games made by homebrew developers, edit yer saves, 'n much, much more.

    Wha' be Custom Firmware?

    Custom Firmware ("CFW") enables ye t' use more advanced hacks that any normal homebrew can nah acheive.

    Wha' does this guide install?

    This guide, will install either CFW, or Th' Homebrew Launcher

    Wha' can I do wit' Custom Firmware?

    • Install games fer free much easier than previously possible
    • Protect against bricks by makin' th' console use yer SD card/USB as its internal memory
    • Play games fer older systems by usin' emulators
    • Install homebrew titles t' yer system, 'n 'ave them appear on yer Home Menu
    • Run many ole Nintendo DS flash carts that were blocked long ago or ne'er worked.

    Wha' do I needs t' know afore startin'?

    • Every time ye modify yer system, thar be a chance that yer console will brick! Precautions are taken so that 'tis unlikely t' happen, but th' risk be present.
    • Keep yer device plugged in AT ALL TIMES DURING THIS PROCESS t' avoid sudden power loss!
